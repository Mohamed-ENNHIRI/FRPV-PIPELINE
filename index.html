§ 5 · ROOF SEGMENTATION 06 / 07

From a 3D point cloud to roof planes.

For every PV-positive building in a tile, the engine isolates the roof points, fits flat regions, and emits one tilt/azimuth/area record per region — all in a JSON file per tile.

1Clip building points to cadastre
2Voxelize on a 1 m grid
3Triangulate & filter slivers
4Region-grow by surface normal
5Merge near-coplanar regions
6Project onto mean plane → JSON
